# Shitla Mandir Website 🌸

This is a simple, devotional website dedicated to **Shitla Mandir**, showcasing the temple’s festivals, daily puja timings, and guidelines for devotees.

## Features

- **Event Timeline:** Sheetala Ashtami dates and other important events.  
- **Daily Puja Timings:** Morning and evening puja schedules.  
- **Guidelines for Visitors:** Tips and instructions for devotees visiting the temple.  
- **Responsive Design:** Works on both desktop and mobile devices.  
- **Decorative & Aesthetic:** Flower-style emojis and neat formatting for better readability.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (optional for animations)
